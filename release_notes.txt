# v2.3.2 - Entry Spam Moderation And Dashboard Status Updates

## Overview
SyntekPro Forms v2.3.2 improves entry moderation by adding a dedicated Spam status workflow across the entries list and reading pane, plus updated dashboard visibility for Spam counts.

## Added
- `Spam` as a first-class entry status in admin entry management.
- Entry reading pane controls to toggle between `Mark as Spam` and `Not Spam`.
- Entries list support for:
  - Spam status tab and status filter
  - Row-level spam toggle actions
  - Bulk action: `Recover from Spam`
- Dashboard widget Spam summary card with direct link to filtered Spam entries.

## Changed
- Unified status update path for entries via shared status update action (`read`, `unread`, `spam`).
- Opening an entry now auto-marks as read only when current status is `unread`.
- Dashboard read counter now uses explicit `read` status count.
- Plugin version bumped to 2.3.2.
- DB version constant bumped to 2.3.2.
