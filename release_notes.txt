# v2.3.3 - Entries UI And Modal Redesign

## Overview
SyntekPro Forms v2.3.3 delivers a complete visual overhaul of the Entries admin experience, including a redesigned entries table, toolbar, and entry detail modal for a more modern and polished workflow.

## Added
- New entries page visual theme:
  - Refined header and stats presentation
  - Modern status tabs and filter/action toolbar styling
  - Improved table readability, status clarity, and action button presentation
- Redesigned entry detail modal:
  - Dedicated toolbar and status chip
  - Metadata card grid for key submission info
  - Improved captured-fields layout and notes section styling
- Better responsive treatment for entries screens on smaller devices.

## Changed
- Refactored entry modal rendering to class-based markup in admin JS for maintainability and consistent theming.
- Added `Read` metric card to entries overview stats.
- Plugin version bumped to 2.3.3.
- DB version constant bumped to 2.3.3.
